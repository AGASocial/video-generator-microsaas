# Theme System

This folder contains the theme system for the application. The theme system is designed to be modular and extensible, allowing you to easily create and switch between different themes.

## Structure

```
themes/
├── types.ts          # TypeScript type definitions (shared across all themes)
├── index.ts          # Main entry point exporting all theme utilities
├── README.md         # This file
├── default/          # Default theme
│   ├── index.ts      # Default theme configuration (light/dark colors, radius, fonts)
│   ├── fonts.ts      # Font configuration and class names
│   └── constants.ts  # CSS class constants and utility combinations
├── christmas/        # Christmas theme (future)
├── dark/             # Dark theme variant (future)
└── light/            # Light theme variant (future)
```

Each theme folder contains:
- **`index.ts`** - Theme configuration (colors, radius, fonts)
- **`fonts.ts`** - Font configuration specific to the theme
- **`constants.ts`** - CSS class constants specific to the theme

## Usage

### Using Theme Constants

```tsx
import { cssClasses, classCombinations } from '@/themes'

// Use predefined class combinations
<div className={classCombinations.card}>
  <h1 className={cssClasses.heading1}>Title</h1>
</div>
```

### Using Font Classes

```tsx
import { fontClasses } from '@/themes'

<h1 className={`${fontClasses.sans} ${fontClasses.bold} ${fontClasses['3xl']}`}>
  Heading
</h1>
```

### Creating a New Theme

1. Create a new folder in the `themes` directory (e.g., `themes/ocean/`)
2. Create the theme files:
   - `index.ts` - Theme configuration (required - must export default)
   - `fonts.ts` - Font configuration (optional, can import from default)
   - `constants.ts` - CSS constants (optional, can import from default)

3. Example theme structure:

```typescript
// themes/ocean/index.ts
import type { ThemeConfig } from '../types'
import { defaultTheme } from '../default'

const oceanTheme: ThemeConfig = {
  ...defaultTheme,
  name: 'ocean',
  colors: {
    light: {
      ...defaultTheme.colors.light,
      // Override colors for light mode
      primary: 'oklch(0.5 0.2 220)', // Ocean blue
      // ... other color overrides
    },
    dark: {
      ...defaultTheme.colors.dark,
      // Override colors for dark mode
      // ...
    },
  },
}

// Export as both named and default for compatibility
export { oceanTheme }
export default oceanTheme
```

4. **The theme will be automatically detected!** The registry is auto-generated by scanning the `themes/` directory. Just run:
   ```bash
   npm run generate:themes
   ```
   
   Or it will run automatically before build (`npm run build`).

**Note**: Each theme folder must have an `index.ts` file that exports a default `ThemeConfig` for it to be automatically included in the registry.

## Current Theme

The default theme uses:
- **Primary Font**: Inter (modern, clean, highly readable)
- **Monospace Font**: JetBrains Mono (developer-friendly)
- **Color System**: OKLCH color space for better color consistency
- **Border Radius**: 0.625rem (10px)

## Font Configuration

The app uses Inter as the primary font, which is:
- Modern and clean
- Highly readable at all sizes
- Optimized for screens
- Excellent for UI/UX applications

Fonts are loaded via Next.js font optimization for optimal performance.
